# proyecto-AJAX
Esta es una tienda virtual desarrollada con AJAX, creada durante mis primeras etapas de aprendizaje en desarrollo web. El proyecto demuestra cómo integrar peticiones asíncronas para actualizar información en tiempo real sin recargar la página, ofreciendo una experiencia más fluida y dinámica para el usuario.

Aunque es un proyecto inicial, incluye conceptos clave como manejo de datos, interacción con el servidor y construcción de interfaces responsivas, lo que lo convierte en una base sólida para seguir avanzando en el desarrollo frontend y backend.
